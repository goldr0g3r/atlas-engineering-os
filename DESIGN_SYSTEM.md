# Atlas visual system

The redesign follows the supplied references without copying logos, names, people, or product imagery.

- Light frosted workspace on a warm-to-cool ambient background
- Compact icon rail on desktop and bottom navigation on mobile
- Rounded cards, restrained shadows, pill controls, and high-density data tables
- Project dashboard with KPI cards, recent projects, and module shortcuts
- BOM detail header, summary metrics, component table, and right-side add-component drawer
- Minimum 44px interactive targets, visible focus states, semantic table markup, and responsive overflow
